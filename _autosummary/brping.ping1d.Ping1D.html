brping.ping1d.Ping1D

class brping.ping1d.Ping1D

Bases: PingDevice

Represents a Ping1D Echosounder from Blue Robotics.

This class provides methods to control and retrieve data from the Ping1D echosounder, which includes functionality to measure distances and read from underwater environments.

get_distance()

Get a distance message from the device. Message description: The distance to target with confidence estimate. Relevant device parameters during the measurement are also provided.

Returns:

None if there is no reply from the device, otherwise a dictionary with the following keys:

  • distance: The current return distance determined for the most recent acoustic measurement. (Units: mm)

  • confidence: Confidence in the most recent range measurement. (Units: %)

  • transmit_duration: The acoustic pulse length during acoustic transmission/activation. (Units: us)

  • ping_number: The pulse/measurement count since boot.

  • scan_start: The beginning of the scan region in mm from the transducer. (Units: mm)

  • scan_length: The length of the scan region. (Units: mm)

  • gain_setting: The current gain setting. 0: 0.6, 1: 1.8, 2: 5.5, 3: 12.9, 4: 30.2, 5: 66.1, 6: 144

rtype:

dict or None

get_profile()

Get a profile message from the device. Message description: A profile produced from a single acoustic measurement. The data returned is an array of response strength at even intervals across the scan region. The scan region is defined as the region between <scan_start> and <scan_start + scan_length> millimeters away from the transducer. A distance measurement to the target is also provided.

Returns:

None if there is no reply from the device, otherwise a dictionary with the following keys:

  • distance: The current return distance determined for the most recent acoustic measurement. (Units: mm)

  • confidence: Confidence in the most recent range measurement. (Units: %)

  • transmit_duration: The acoustic pulse length during acoustic transmission/activation. (Units: us)

  • ping_number: The pulse/measurement count since boot.

  • scan_start: The beginning of the scan region in mm from the transducer. (Units: mm)

  • scan_length: The length of the scan region. (Units: mm)

  • gain_setting: The current gain setting. 0: 0.6, 1: 1.8, 2: 5.5, 3: 12.9, 4: 30.2, 5: 66.1, 6: 144

  • profile_data: An array of return strength measurements taken at regular intervals across the scan region.

rtype:

dict or None
