parse_byte(msg_byte)

Returns the current parse state after feeding the parser a single byte.

‘msg_byte’ is the byte to parse.

If it completes a valid message, returns PingParser.NEW_MESSAGE. The decoded PingMessage will be available in the self.rx_msg attribute

until a new message is decoded.
