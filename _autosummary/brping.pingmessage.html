brping.pingmessage

Python implementation of the Blue Robotics ‘Ping’ binary message protocol.

This module encapsulates the structure and functionality required to create, parse, and manage messages used in the Blue Robotics ‘Ping’ protocol, specifically designed for communication with underwater sonar devices.
