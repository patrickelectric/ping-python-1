brping.pingmessage.PingMessage

class brping.pingmessage.PingMessage(msg_id=0, msg_data=None)

Bases: object

Represents a message protocol for the Blue Robotics ‘Ping’ devices.

This class provides methods to pack and unpack message data into/from binary formats, handle checksums, and facilitate the communication between client applications and the device.

Parameters:

  • msg_data – Binary data buffer for unpacking into a message object. If None, an empty message is created.
